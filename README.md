# 🏉 Rugby Performance — Déploiement GitHub Pages

## 3 étapes pour avoir l'app sur ton iPhone

### Étape 1 — Créer le dépôt GitHub
1. Va sur **github.com** → clique sur **"New"** (bouton vert)
2. Nom du dépôt : `rugby-perf`  
3. Coche **"Public"**
4. Clique **"Create repository"**

### Étape 2 — Uploader les 2 fichiers
1. Dans ton nouveau dépôt, clique **"Add file" → "Upload files"**
2. Glisse-dépose **`index.html`** et **`manifest.json`**
3. Clique **"Commit changes"**

### Étape 3 — Activer GitHub Pages
1. Dans le dépôt → clique **"Settings"** (onglet en haut)
2. Colonne gauche → **"Pages"**
3. Sous "Source" → sélectionne **"Deploy from a branch"**
4. Branch : **"main"** → dossier **"/ (root)"** → **"Save"**

⏳ Attends 2-3 minutes, puis ton URL est :
**`https://TON-USERNAME.github.io/rugby-perf`**

---

## Installer sur l'écran d'accueil iPhone

1. Ouvre l'URL dans **Safari** (pas Chrome, pas Firefox)
2. Appuie sur l'icône **Partager** (carré avec flèche vers le haut)
3. Scroll vers le bas → **"Sur l'écran d'accueil"**
4. Appuie **"Ajouter"**

L'app apparaît comme une vraie app iPhone avec l'icône 🏉

## Fonctionnalités

- 🏋 **Séance interactive** — validation série par série, poids/reps/RPE par série
- 📈 **Patterns** — Plat, Montée, Descente, Pyramide, Drop set
- ⏱ **Timers** — repos automatique + chronomètre libre
- 🤖 **IA** — prédiction des poids, analyse des notes, optimisation séance
- 🥗 **Nutrition** — suivi macros, recherche Open Food Facts, aliments favoris
- 🎵 **Deezer** — accès rapide à ta playlist depuis la séance
- 🧍 **Posture** — routine quotidienne complète
- 💾 **Import/Export** — sauvegarde JSON sur iPhone
- 📱 **PWA** — fonctionne hors-ligne une fois chargée
